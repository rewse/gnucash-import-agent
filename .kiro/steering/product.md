# Product Overview

GnuCash Statement Importer - A tool for importing online financial statements into GnuCash PostgreSQL database.

## Purpose

Automate the import of transaction data from various online sources (banks, credit cards, prepaid cards, etc.) into GnuCash accounting software.

## Key Features

- Browser automation for statement retrieval
- Transaction parsing and account mapping
- SQL generation for direct database insertion
- Duplicate detection before insertion
